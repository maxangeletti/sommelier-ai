## 🎯 Features

### Suggestion Mode for Generic Queries
**Issue**: Query generiche come "amaro" restituivano risultati con substring match non desiderati (Negroamaro, Amarone).

**Solution**: 
- Nuovo sistema suggestion mode per query ≤5 caratteri senza intent specifico
- Funzione `is_generic_query()` - identifica query ambigue
- Endpoint `/search` ora ritorna `{results: [], suggestions: [...]}` per query generiche
- Fuzzy matching semplificato (dist < 3 per suggestions)

**Example**:
```bash
Query: "amaro"
Before: [Amarone €58, Negroamaro €12]  # risultati diretti
After:  {results: [], suggestions: ["Amarone"]}  # suggestion mode
```

**Files Modified**:
- `backend/main.py`:
  - Line 125: `fuzzy_match_query()` - semplificato logic (dist < 3)
  - Line 2812: `is_generic_query()` - NEW function
  - Line 2841: Suggestion check in `/search` endpoint
